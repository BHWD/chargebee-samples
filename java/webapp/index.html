Checkout existing subscriptions

This tutorial will help us implement a checkout flow for existing customers to upgrade/downgrade their subscriptions. Using ChargeBee's hosted checkout pages, we can allow users to update their subscription by changing their plans, addons and product quantity. The sample application shows how an existing trial user upgrades to a paid subscription from within a merchant application.

Checkout using Stripe.js & ChargeBee

The intent of this tutorial is to help merchants using Stripe as their payment gateway to build a custom checkout page using Stripe.js. Using Stripe's JavaScript on the checkout form, your servers do not handle any credit card information reducing your need to worry about PCI compliance. Our sample application, at a high-level, shows the checkout flow starting from the merchant site to the success page.
