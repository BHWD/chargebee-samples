Checkout a new subscription

An easy tutorial to build a simple checkout flow for users to sign up for a subscription using ChargeBee's hosted checkout pages. This tutorial is a primer for using ChargeBee's hosted page API. You can extend this and go on to build complex multi-step checkout flow.

Checkout an existing subscription

This tutorial is quite similar to "Checkout New Subscription" but explains the checkout implementation for existing customers using ChargeBee's hosted checkout pages. The sample application in this tutorial shows how an existing trial user upgrades to a paid subscription from within a merchant application.

Checkout using Stripe.js

This tutorial walks you through the steps involved in using Stripe.js and ChargeBee's API to build custom checkout pages for creating subscriptions. Stripe's JavaScript ensures that sensitive credit card information is not passed to your server thereby reducing your PCI compliance requirements.
Note: Stripe.js can be used only if your payment gateway is Stripe.
